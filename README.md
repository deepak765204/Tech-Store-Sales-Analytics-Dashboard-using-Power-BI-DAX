# Tech-Store-Sales-Analytics-Dashboard-using-Power-BI-DAX
Developed an interactive sales analytics dashboard using Power BI and DAX to analyze 50M+ revenue data, enabling insights into product performance, regional trends, and customer behavior.

📊 Tech Store Sales Analytics Dashboard (Power BI + DAX)

📌 Project Overview

This project presents an interactive Sales Analytics Dashboard developed using Power BI to analyze and visualize the performance of a tech retail store.

The dashboard provides deep insights into sales, customer behavior, product performance, and regional trends, helping in data-driven decision-making.

🎯 Objectives
->Analyze overall business performance
->Track sales trends across time
->Identify top-performing products
->Compare online vs offline sales
->Understand customer and payment behavior

🛠️ Tools & Technologies
->Power BI
->DAX (Data Analysis Expressions)
->Data Visualization Techniques
->Excel / CSV Dataset

📌 KPI Metrics
->Total Sales: 51M
->Total Orders: 1000
->Total Customers: 996
->Total Quantity Sold: 4899
->Online vs Non-Online Deliveries

📈 Sales Analysis
->Sales by Product
->Sales by State & City
->Monthly Sales Trend
->Sales by Channel (Online / Store / Direct)

🌍 Geographical Insights
->Interactive map visualization of sales across locations
->Region-wise performance analysis

💳 Customer & Payment Analysis
->Payment method distribution
->Customer-level insights

Product Insights
->Top-selling products
->Product-wise quantity and revenue contribution

⚙️ DAX Implementation

Used DAX formulas to create:

->Calculated measures (Total Sales, Average Unit Price)
->KPI indicators
->ggregations and dynamic calculations

▶️ How to Use
->Download the .pbix file
->Open in Power BI Desktop
->Use filters and slicers to explore insights

🎯 Key Insights
->Identified top-performing products contributing to revenue
->Observed monthly fluctuations in sales
->Compared effectiveness of online vs offline sales channels
->Analyzed regional sales performance

📚 Learning Outcomes
->Hands-on experience with Power BI dashboard development
->Strong understanding of DAX calculations
->Improved data storytelling and visualization skills
